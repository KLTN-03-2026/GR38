# AI History Learning Assistant 🇻🇳
Hệ thống hỗ trợ ôn tập lịch sử Việt Nam tích hợp AI giúp học tập hiệu quả hơn.

## 🚀 Công nghệ
- Frontend: React + Vite
- Backend: Node.js / Express
- Database: MongoDB
- AI: Gemini API

## ⚙️ Cài đặt

### 1. Clone repo
git clone https://github.com/KLTN-03-2026/GR38.git

### 2. Frontend
cd frontend/ai-history-learning
npm install
npm run dev

### 3. Backend
cd backend
npm install
npm start

## ✨ Tính năng
- Hỏi đáp lịch sử bằng AI
- Tạo quiz tự động
- Lưu tiến trình học
